# Kado24 Merchant App

Flutter mobile application for merchants to manage vouchers and redeem customer QR codes.

## Features

- Merchant authentication
- QR code scanner for redemption
- Voucher management (create, edit, pause)
- Sales dashboard
- Transaction history
- Payout tracking

## Setup

```bash
flutter pub get
flutter run
```

## Key Features

### QR Scanner
- Scan customer voucher QR codes
- Validate voucher
- Process redemption
- Real-time confirmation

### Dashboard
- Sales metrics
- Recent transactions
- Payout status
- Voucher performance

### Voucher Management
- Create new vouchers
- Edit existing vouchers
- Pause/activate vouchers
- Stock management

## Backend Integration

Connects to:
- Auth Service (8081)
- Merchant Service (8088)
- Voucher Service (8083)
- Redemption Service (8087)
- Payout Service (8092)
